# MSS Website

## Development

### Running Locally

 1. Install `yarn`
 2. Run `git clone`
 3. Navigate into `msswesbite`
 4. Run `yarn`
 5. Run `yarn serve`
 6. Navigate to `localhost:8080` and the website should be displayed!
